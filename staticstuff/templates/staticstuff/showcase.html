bowdoin.wine

bowdoin.wine is the home of Bowdoin College's wine club. Due to the club being about wine, it is not officially chartered. As such, I needed a place to communicate with club members, host surveys, and more.

Features:

  • Showcase of the wine tastings we've done.
  • Showcase of the wines we have drank.
  • Searchable database of members' tasting notes on said wines. (anonymized if club members are logged in or not)
  • A survey hosting mechanism for when I need input from club members

Technologies:

  • Django - I have found Django to be very enjoyable to work with and as this is an application that needs auth+CRUD, I found it to be a very nice fit.
  • Python - By virtue of django.
  • Postgres - In dev, I am using SQLite, but in prod I have deployed a postgres database.
  • Gunicorn - As the web server to host bowdoin.wine
  • Nginx - As the web server that "faces" the outside world on my VPS. I've chosen file based sockets to communicate between Nginx and Gunicorn.
  • Debian Linux - As the base of my hosting stack.

Dactyl 30

The Dactyl30 is my modification of the Dactyl manuform keyboard. It is an ultra-minimalist, ergonomic keyboard that is optimized to the curvature of my hands.

Technologies

  • Python - I modified some python scripts that generate an OpenSCAD model and made further modifications to the housing with Blender.
  • C - Due to the limited key layout, I had to use embedded C to create a custom key layout that involved multiple key layers and chording.
  • Blender - To modify the generated CAD files.
  • atmega32u4 - The main cpu chip I used to drive the keyboard.
